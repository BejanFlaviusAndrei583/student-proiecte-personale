# Enterprise Network Project (Cisco Packet Tracer)

## About

This project was completed as part of my Cisco networking training using Cisco Packet Tracer.

The goal of the project was to practice the networking concepts learned during the course by configuring a small enterprise network consisting of multiple locations.

---

## What I Practiced

- Basic router and switch configuration
- VLAN configuration
- Inter-VLAN Routing
- Static Routing
- DHCP
- SSH remote access
- Basic wireless network configuration
- Basic switch security
- Testing network connectivity

---

## Tools Used

- Cisco Packet Tracer
- Cisco IOS CLI

---

## Network Overview

The network includes four simulated locations:

- Home Office
- Branch Office
- Central Office
- Data Center

Each location was configured to communicate with the others while using different VLANs and IP networks.

---

## What I Learned

Working on this project helped me better understand:

- IP addressing
- VLANs
- Routing between networks
- Switch configuration
- Basic network troubleshooting
- Enterprise network structure
